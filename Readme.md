# 📊 Wi-Fi Usage and Student Productivity Analysis

## Overview
This project analyzes how Wi-Fi usage and behavioral factors relate to productivity using survey data. It includes data collection, cleaning, exploratory analysis, visualization, and linear regression modeling.

## Objectives
- Study relationship between Wi-Fi usage and productivity  
- Perform exploratory data analysis  
- Build a simple predictive model  
- Derive insights from behavioral data  

## Dataset Features
- Age  
- Wi-Fi Usage Hours  
- Study Hours  
- Instagram Usage  
- Sleep Hours  
- Productivity Score  

## Methods Used
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Data Visualization / Dashboarding  
- Linear Regression Modeling  

## Key Insights
- Study hours showed positive association with productivity  
- Wi-Fi usage showed positive relationship with productive activities  
- Sleep showed notable influence in the model  
- Results suggest productivity depends on multiple factors  

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn
